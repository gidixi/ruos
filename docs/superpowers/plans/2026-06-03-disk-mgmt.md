### Task 1: `umount` (vfs::unmount + host fn + tool) — the important one

**Files:** `kernel/src/vfs/mod.rs`, `kernel/src/wasm/host/proc.rs`, `user/umount/` (new), `Makefile`, `limine.conf`.

Read `kernel/src/vfs/mod.rs` (`MOUNTS`, `mount`, `is_mounted`), and `ruos_chdir` in `proc.rs` (the `guest_read(path_ptr, path_len)` pattern + `func_wrap` registration + the `VfsError`→errno mapping).

- [ ] **Step 1: `vfs::unmount`** (mod.rs):
```rust
/// Unmount the filesystem at exactly `prefix` (e.g. "/mnt"). Dropping the
/// FsImpl releases the backing device (the SATA port) once no open file still
/// references it. Refuses "/" (the root tmpfs).
pub fn unmount(prefix: &str) -> Result<(), VfsError> {
    if prefix == "/" { return Err(VfsError::InvalidPath); }
    let mut m = MOUNTS.lock();
    let before = m.len();
    m.retain(|(p, _)| p != prefix);
    if m.len() == before { Err(VfsError::NotFound) } else { Ok(()) }
}
```

- [ ] **Step 2: `ruos_umount` host fn** (proc.rs), mirror `ruos_chdir`:
```rust
fn ruos_umount(caller: Caller<'_, RuntimeState>, path_ptr: i32, path_len: i32) -> Result<i32, Error> {
    let buf = match crate::wasm::host::mem::guest_read(&caller, path_ptr, path_len) { Ok(b)=>b, Err(_)=>return Ok(-1) };
    let path = match core::str::from_utf8(&buf) { Ok(s)=>s, Err(_)=>return Ok(-1) };
    match crate::vfs::unmount(path) {
        Ok(()) => Ok(0),
        Err(crate::vfs::VfsError::InvalidPath) => Ok(-2),
        Err(_) => Ok(-1),   // NotFound / not mounted
    }
}
```
Register `.func_wrap("ruos","umount", ruos_umount)?`. (Match the exact `Caller`/store/`Error` types + the `guest_read` return shape `ruos_chdir` uses.)

- [ ] **Step 3: `user/umount/` tool** (mirror a thin tool like `user/mkdir` or `user/install`): import `#[link(wasm_import_module="ruos")] extern "C" { fn umount(ptr:i32, len:i32)->i32; }`. `main()`: require one arg (the path); call `umount(p.as_ptr() as i32, p.len() as i32)`; map `0`→`umount: <path> unmounted`, `-2`→`umount: cannot unmount <path>`, else `umount: <path> not mounted` + exit 1. Add `umount` to `BIN_TOOLS` (Makefile) + a `/bin/umount.wasm` `limine.conf` module entry (copy an existing tool's 2 lines). Commit the built `user-bin/umount.wasm` blob (repo convention).

- [ ] **Step 4: build + in-guest verify (the proof it unblocks install).**
`make iso` clean; `umount.wasm` in `/bin`. Then boot with a disk M1 auto-mounts at `/mnt` (build a GPT disk like `tests/gpt-test.sh` does, OR run `mkboot 64` on a blank disk then reboot so `/mnt` is mounted) + an init that does:

WIPING` / `install: ok` (NOT `install: /mnt is mounted, refusing`). That sequence proves umount freed the port + the guard now passes. (Write a small throwaway check script staged on ext4 /tmp for any mtools; here it's serial-only.) Also `make run-test` → `TEST_PASS` (umount unused there).
If install STILL refuses after umount: the MOUNTS entry wasn't removed or a file holds the port — report the serial.

### Task 2: `disks` (clean listing) + `install`-no-arg cleanup

**Files:** `kernel/src/wasm/host/proc.rs`, `user/disks/` (new), `user/install/src/main.rs`, `Makefile`, `limine.conf`.

- [ ] **Step 1: `ruos_sata_list(buf_ptr, buf_cap) -> i32`** (proc.rs): build a `String`: for each `idx` in `crate::ahci::sata_ports()`, `acquire_port(idx)` → `let _ = write!(s, "{}\t{:?}\t{} MiB\n", idx, p.model, p.sectors/2048);` (skip if `acquire_port` is None). Write the UTF-8 bytes into the guest buffer at `buf_ptr` (≤ `buf_cap`), exactly as `ruos_pci_list` writes its listing; return the byte count (or `-1` if `s.len() > buf_cap`).
